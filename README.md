# FocusDice

Идея: мини-приложение против прокрастинации. Пользователь нажимает кнопку и получает маленькое, случайное, выполнимое действие: 25 минут фокуса, глоток воды, уборка стола, короткая прогулка, разбор почты и т.д.

## Что внутри
- один экран
- случайный выбор мини-задачи
- счётчик бросков
- счётчик выполненных действий
- сохранение статистики через SharedPreferences
- аккуратный UI без сложных зависимостей

## Как собрать APK
1. Открой проект в Android Studio Hedgehog или новее.
2. Дождись загрузки Gradle.
3. Если Android Studio предложит установить Android SDK 34, согласись.
4. Выбери `Build > Build Bundle(s) / APK(s) > Build APK(s)`.
5. Готовый debug APK появится примерно в `app/build/outputs/apk/debug/app-debug.apk`.
